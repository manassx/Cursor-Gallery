# CursorGallery

Interactive photo galleries where images follow your cursor movement.

## About

CursorGallery transforms static photo collections into dynamic, cursor-driven experiences. Move your mouse across the
canvas and watch as images appear and fade in a trailing effect, creating an immersive way to explore memories.

## Features

- **Cursor-Trail Gallery** - Images respond to mouse movement
- **Adjustable Sensitivity** - Control how images appear
- **Fullscreen Lightbox** - Click any image for detailed view
- **Dark/Light Themes** - Switch between modes
- **Drag & Drop Upload** - Easy image management

## Tech Stack

- React 18
- Vite
- Framer Motion
- React Router
- Zustand
- Lucide Icons
